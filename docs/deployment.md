## 环境要求

### 系统要求
- Python 3.9+
- SQLite 3
- 磁盘空间：建议10GB以上
- 内存：建议≥1GB
- CPU：建议≥1核
- 网络：带宽≥2Mbps，稳定连接
- 操作系统：Linux (推荐 Ubuntu 20.04+)

所有依赖包已在 `requirements.txt` 中列出。
